Overview

The game involves at least two players.

Players take turns to guess letters in a phrase.

Points are awarded for correct guesses based on the current letter value.

The game continues until the phrase is solved.

  • solved: A boolean flag indicating if the game's phrase has been solved.
  • currentPlayer: An integer tracking the current player (1 or 2).
  • game: An object representing the game, with methods to handle game logic.

Game Flow

correct is initially set to true.

Game enters a loop that continues until solved is true.

Game Loop:

Inside the loop, a nested loop runs as long as correct is true.

The game updates and displays the letter value.

The current player is prompted to make a guess.

Guess Handling:

The player's guess is read from input (sc.nextLine()).

The game checks if the guess solves the phrase (game.isSolved(guess)).

The game checks if the guessed letter is correct (game.guessLetter(guess)).

Point Allocation:

Player Switching:

If a guess is incorrect (correct is false), control switches to the other player.

End of Game:

Once the phrase is solved (solved is true), the loop ends.

The game displays the final points for both players.
