Overview

The game involves at least two players, where players take turns to guess letters in a phrase. Points are awarded for correct guesses based on the current letter value. The game continues until the phrase is solved.

Variables
  • game: Object for game logic.
  • player1, player2: Objects representing the players.
  • currentPlayer: Integer tracking the current player (1 or 2).
  • correct: Boolean flag for last guessed letter's correctness.
  • solved: Boolean flag for the game's phrase being solved.
Game Flow
  1. Initialization: Setting 'correct' to true and entering the main game loop.
  2. Game Loop: Continues until 'solved' is true.
  3. Guess Handling: Reading and evaluating player's guess.
  4. Point Allocation: Awarding points for correct guesses.
  5. Player Switching: Alternating between players based on guesses.
  6. End of Game: Displaying final points and concluding the game.
Methods Used
  • game.setLetterValue(): Sets the current letter value.
  • game.getSolvedPhrase(): Returns the current state of the solved phrase.
  • game.getLetterValue(): Returns the current letter value.
  • game.isSolved(): Checks if the phrase is solved.
  • game.guessLetter(): Checks if the guessed letter is in the phrase.
  • player.getName(): Returns player's name.
  • player.addToPoints(): Adds points to the player's score.
  • player.getPoints(): Returns the player's current points.
